## Knowledge Base
### Fuzzy Logic Calculator
Web-based fuzzy logic calculator. Created based on the research done by Cynthia Taylor et al., and developed for traffic engineering students and enthusists to learn how fuzzy logic is applied in freeway operations.

### LOS Calculator
Web-based, HCM-like calculator for calculating Level of Service of facilities using methods described in HCM 6th Edition. More to come...

### Queue Visual
Web-based tool for visualizing forming of queues on freeway, ramps, arterials, etc. under user-defined conditions. More to come...
